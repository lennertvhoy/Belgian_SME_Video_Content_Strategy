# Visual Brand Guidelines & Templates
## Consistent Design System voor Belgische KMO Video Content

### 🎨 Brand Identity Framework

#### **Core Brand Elements**
```
PRIMARY LOGO:
├── Format: Vector (SVG) + High-res PNG
├── Colors: Full color, white, black versions
├── Placement: Lower right corner (standard)
├── Size: 10% of video width maximum
└── Buffer: 5% margin from edges

TYPOGRAPHY SYSTEM:
├── Primary: Segoe UI (Microsoft ecosystem alignment)
├── Secondary: Open Sans (web compatibility)
├── Headers: Bold weight, 32-48pt
├── Body: Regular weight, 24-30pt
└── Captions: Light weight, 18-22pt

COLOR PALETTE:
├── Primary Blue: #0078D4 (Microsoft Azure brand)
├── Secondary Blue: #106EBE (darker variation)
├── Success Green: #107C10 (positive messaging)
├── Warning Orange: #FF8C00 (attention/urgency)
├── Error Red: #D13438 (problems/risks)
├── Neutral Gray: #605E5C (text/backgrounds)
└── Background: #FAFAFA (subtle off-white)
```

#### **Visual Style Principles**
```
CLEAN & PROFESSIONAL:
├── Minimal text per slide (6 words maximum)
├── High contrast ratios (4.5:1 minimum)
├── Consistent spacing (8px grid system)
└── Professional photography/illustrations only

ACCESSIBILITY FOCUSED:
├── WCAG 2.1 AA compliance
├── Colorblind-friendly palette
├── Large, readable fonts
├── Audio descriptions available
└── Subtitle/caption support

TECHNICALLY ACCURATE:
├── Real screenshots (no mockups for technical content)
├── Updated software versions
├── Belgian-specific settings/configurations
└── Accurate pricing/dates
```

### 📐 Video Template Specifications

#### **Standard Format Templates**

**How-To Videos (10 minutes)**
```
INTRO SECTION (0-30 seconds):
├── Background: Subtle gradient (Primary Blue to white)
├── Logo: Lower right corner
├── Title: Large text, center-aligned
├── Subtitle: Problem statement, smaller text
└── Duration indicator: "10 minuten tutorial"

MAIN CONTENT (30 seconds - 9 minutes):
├── Screen recording: Full screen or windowed
├── Overlay graphics: Minimal, contextual
├── Progress indicator: Subtle progress bar
├── Section transitions: 2-second fade
└── Callout boxes: Yellow highlight for key points

OUTRO SECTION (9-10 minutes):
├── CTA slide: Clear action items
├── Contact information: Website, phone, email
├── Related videos: Thumbnail grid
├── Subscribe reminder: Animated button
└── Logo: Prominent placement
```

**Mini-Workshop Videos (2 minutes)**
```
RAPID-FIRE FORMAT:
├── Background: Consistent brand colors
├── Text overlays: Large, bold numbers
├── Icon system: Consistent style library
├── Transitions: Quick cuts (0.5 second max)
└── Progress: Numbered sections (1/5, 2/5, etc.)

VISUAL HIERARCHY:
├── Level 1: Tip numbers (48pt, bold)
├── Level 2: Tip titles (36pt, regular)
├── Level 3: Descriptions (24pt, light)
├── Level 4: Details/prices (18pt, highlighted)
└── Level 5: Disclaimers (14pt, gray)
```

### 🎭 Animation Style Guide

#### **Standard Animations**
```
ENTRANCE EFFECTS:
├── Fade In: 0.3 seconds, ease-out
├── Slide In: From right, 0.5 seconds
├── Scale Up: From 0.8x to 1.0x, 0.4 seconds
├── Reveal: Mask wipe effect, 0.6 seconds
└── Type Effect: Character-by-character, 0.05s per char

EMPHASIS EFFECTS:
├── Pulse: 1.2x scale for 0.2 seconds
├── Color Change: 0.3 second transition
├── Glow: Subtle shadow increase
├── Bounce: Small vertical movement
└── Highlight: Background color flash

TRANSITION EFFECTS:
├── Cross Fade: Between scenes, 1 second
├── Slide Transition: Scene wipe, 0.8 seconds
├── Zoom: Focus change, 0.6 seconds
├── Split Screen: Content comparison
└── Picture-in-Picture: Demo + presenter
```

#### **Motion Graphics Principles**
```
TIMING & EASING:
├── Fast actions: 0.2-0.3 seconds
├── Medium actions: 0.4-0.6 seconds
├── Slow actions: 0.8-1.2 seconds
├── Easing: Ease-out for natural feel
└── Stagger: 0.1 second delay between elements

MOVEMENT PATTERNS:
├── Horizontal: Left-to-right reading pattern
├── Vertical: Top-to-bottom importance
├── Circular: Process flows and cycles
├── Diagonal: Dynamic energy/growth
└── Static: Stability and trust
```

### 📱 Platform-Specific Adaptations

#### **YouTube Optimization**
```
THUMBNAIL DESIGN:
├── Size: 1280×720 pixels (16:9 ratio)
├── Text: Maximum 6 words, 60pt minimum
├── Faces: Eye contact, emotional expression
├── Contrast: High contrast background
├── Branding: Logo in consistent corner
└── A/B Test: Multiple versions per video

VIDEO SPECS:
├── Resolution: 1920×1080 (1080p minimum)
├── Frame Rate: 30fps standard
├── Audio: -12dB to -6dB levels
├── Length: 8-12 minutes for how-to content
└── Captions: SRT files uploaded
```

#### **LinkedIn Video Adaptation**
```
NATIVE VIDEO FORMAT:
├── Aspect Ratio: 16:9 (horizontal feed)
├── Duration: 3-5 minutes maximum
├── Captions: Always enabled (85% watch without sound)
├── Hook: First 3 seconds crucial
└── CTA: Clear professional action

SQUARE FORMAT (1:1):
├── Mobile optimization: 1080×1080
├── Text: Larger fonts for mobile viewing
├── Logo: Top or bottom placement
├── Content: Key points only
└── Length: 60-90 seconds maximum
```

#### **Website Integration**
```
EMBEDDED PLAYERS:
├── Responsive design: 16:9 ratio maintained
├── Auto-play: Disabled (user preference)
├── Controls: Standard player controls
├── Quality: Adaptive streaming
└── Analytics: UTM tracking parameters

LANDING PAGE DESIGN:
├── Hero video: Above fold placement
├── Transcript: SEO optimization
├── Related content: Cross-linking
├── Lead capture: Form integration
└── Social sharing: One-click sharing
```

### 📈 Performance Optimization

#### **File Size Management**
```
COMPRESSION SETTINGS:
├── YouTube: H.264, 10-15 Mbps
├── LinkedIn: H.264, 5-8 Mbps
├── Website: H.264, 3-5 Mbps
├── Mobile: H.264, 1-2 Mbps
└── Archive: ProRes (uncompressed backup)

OPTIMIZATION TECHNIQUES:
├── Pre-compression: Render at higher quality
├── Two-pass encoding: Better quality/size ratio
├── Audio compression: AAC, 192-320 kbps
├── Thumbnail optimization: WebP format
└── Progressive download: Fast start playback
```

#### **Loading Performance**
```
WEB OPTIMIZATION:
├── Video preload: Metadata only
├── Thumbnail sprites: Faster scrubbing
├── CDN delivery: Global distribution
├── Adaptive streaming: Quality based on connection
└── Lazy loading: Below-fold videos
```
